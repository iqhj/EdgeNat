**实际使用体验怎么样**

网上有不少用户的测评，整体评价还是不错的。大家比较认可的几个点：

网络速度方面，香港和韩国机房的延迟确实很低。有用户测试香港直连VPS，电信用户ping值稳定在30-40ms，联通和移动也差不多。韩国SK直连的延迟甚至更低一些，晚高峰时段也不会飙升。美国洛杉矶的4837线路，虽然物理距离远，但走的是优化线路,延迟在130-150ms左右，对于美国VPS来说已经相当不错了。

下载速度方面，用户反馈基本能跑满带宽。10M带宽实测下载速度能到1MB/s左右，20M带宽能到2-3MB/s。美国机房因为带宽大，下载大文件的速度会更快。

稳定性是EdgeNat的一大优势。因为用的是企业级硬件加RAID阵列,硬盘故障的概率很低。而且他们24小时有人驻场,出问题能快速响应。根据用户反馈,除了偶尔的维护公告,基本没遇到过长时间宕机的情况。

工单响应速度还行,一般几个小时内会回复。不过EdgeNat没有在线客服,只能提交工单,急事儿的话可能会觉得慢。好在他们家VPS比较稳定,平时用不太需要找客服。

**哪些人适合选EdgeNat**

EdgeNat比较适合这么几类用户：

需要低延迟访问国内网站的海外华人或者留学生,可以选韩国或香港机房。延迟低,访问国内的视频、音乐平台都很流畅。

做外贸或者跨境电商的,需要原生IP的,可以选韩国原生IP或日本双ISP系列。这些IP能解锁当地的流媒体和一些有地域限制的服务,对于做市场调研或者测试很有用。

个人站长建站用,香港直连或美国洛杉矶都不错。香港的优势是延迟低,美国的优势是价格便宜、带宽大。如果网站主要面向国内用户,选香港；如果是英文站或者对带宽要求高,选美国。

需要挂机或者跑一些长期任务的,EdgeNat的稳定性能满足需求。而且他们不限流量的套餐比较多,适合跑一些需要持续传输数据的业务。

**使用EdgeNat需要注意的几个点**

首先,EdgeNat明确禁止代理翻墙业务,购买使用要遵守当地法律法规。虽然技术上你可以在VPS上搭各种服务,但如果被检测到违规使用,会直接封停账号不退款。所以如果你就是想找个梯子,EdgeNat不适合你,别浪费钱。

其次,部分套餐没有数据盘,只有20G系统盘。比如香港直连的2G内存套餐、韩国SK的2G内存套餐,都是只有系统盘没有数据盘。如果你需要存储空间,要么选配置高一点
